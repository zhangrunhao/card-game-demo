# Embracing Vim: The Unsung Hero of Code Editors

Vim 的学习曲线不低，但一旦跨过门槛，它带来的效率提升非常明显。

## 为什么值得投入

1. **键盘驱动**：减少鼠标切换，专注在编辑本身。
2. **心流感**：连续的小动作组合成“宏观操作”。
3. **可塑性**：配置简单且社区资源丰富。

如果你在寻找一款能伴随多年成长的编辑器，Vim 值得尝试。
